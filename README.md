# Menjalankan program
Untuk menjalankan program ini, silahkan change directory ke folder masing-masing tasknya.
Lalu jalankan perintah berikut di terminal:

```go run main.go```

Beberapa task disetting untuk menerima input dari command line arguments. Silahkan ikuti instruksi yang ada di masing-masing task.

Namun, ada task yang sudah disetting dengan inputan default di main functionnya. Silahkan sesuaikan parameter yang ada untuk melakukan pengetesan dengan parameter lainnya.
